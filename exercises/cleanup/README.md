# Cleanup instructions

To clean up the exercise artifacts you can follow these steps:

- Logon to the S/4HANA Cloud system, open Joule, click `New Conversation`
- In SAP Build
  - Navigate within your project into latest deployed version and undeploy your skill from Joule
  - Delete all versions within your project
  - Delete your project
  - Delete your environment



## Summary

The exercise can now be executed by another person!
